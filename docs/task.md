# MeatMetrics 開発タスク一覧 📋

## プロジェクト概要

**アプリケーション名**: MeatMetrics  
**目的**: カーニボア専用の栄養成分計算アプリ  
**主要機能**: ユーザー登録、食材検索、栄養計算、進捗追跡
