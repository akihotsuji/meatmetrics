# MeatMetrics 全体タスク管理 📋

## プロジェクト概要

**アプリケーション名**: MeatMetrics  
**目的**: カーニボア専用の栄養成分計算アプリ  
**主要機能**: ユーザー登録、食材検索、栄養計算、進捗追跡  
**技術スタック**: Spring Boot (Java 17) + React + PostgreSQL + AWS
